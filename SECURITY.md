# Security Policy

## Reporting a Vulnerability

If you discover a security issue with CDD-DesignMD-Pro, please
report it privately via GitHub Security Advisories at:
https://github.com/J0rgw/CDD-DESIGNMD-PRO/security/advisories/new

Do not file a public issue. We aim to respond within 7 days.

## Scope

This skill operates on user-provided code and config files. It
does not exfiltrate code, send data externally, or modify files
outside the project working directory unless explicitly instructed
by the user.
